# 🎯 AI-Powered Resume Screening System

An intelligent resume screening system that uses AI to rank candidates based on job requirements.

## Features

- 📄 **Resume Parsing**: Extracts information from PDF and DOCX resumes
- 🔍 **Smart Search**: Vector-based semantic search using ChromaDB
- 🤖 **AI Ranking**: LLM-powered candidate scoring with customizable weights
- 📊 **Web Interface**: Clean Streamlit UI for easy interaction
- 📥 **Export Results**: Download rankings as CSV or JSON

## Tech Stack

- **Frontend**: Streamlit
- **Vector Database**: ChromaDB
- **LLM**: OpenAI GPT-4o-mini
- **PDF/DOCX Parsing**: PyPDF2, python-docx
- **Embeddings**: Sentence Transformers
